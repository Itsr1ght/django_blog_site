About

This is Example Website for displaying a website with the help of django

Django is a free and open-source web framework that can help to build website with using python

It contains the library that provide the basic server and can easily deployed the website

Django is pretty fast compare to Other frameworks

Django's primary goal is to ease the creation of complex, database-driven websites.

The framework emphasizes reusability and "pluggability" of components, less code, low coupling,

rapid development, and the principle of don't repeat yourself . Python is used throughout,

even for settings, files, and data models.
